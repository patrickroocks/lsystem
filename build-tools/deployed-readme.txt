lsystem
=======

Copyright by Patrick Roocks, mail@p-roocks.de

1) General info
---------------

The lsystem application is an interactive fractal generator for Lindenmayer systems.

https://www.p-roocks.de/wordpress2/lsystem-simulator/
https://github.com/patrickroocks/lsystem

lsystem is licensed under the MIT Licence, see `LICENSE` file for details.
Regarding the Qt libraries, see https://www.qt.io/licensing/ for the license information.

This package was build via the scripts in https://github.com/patrickroocks/lsystem/tree/main/build


2) Run lsystem
--------------

- Extract all the contents of this archive
- Run lsystemapp.exe


3) Settings
-----------

(e.g., configurations of user defined fractals)

They will be stored in config.json in the same folder as lsystemapp.exe
